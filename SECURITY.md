## Reporting a Vulnerability

We take the security of AstroLeads seriously. If you have found a security vulnerability in AstroLeads, please report it to us as described below.

### üìù How to Report
**Do not report security vulnerabilities through public GitHub issues.**

Please send an email to **security@astroleads.com** with:
1. A description of the vulnerability.
2. Steps to reproduce the issue.
3. Potential impact.

### ‚è≥ Response Timeline
We are committed to addressing security issues promptly:
- **Acknowledgement**: Within 24 hours.
- **Assessment**: Within 48 hours.
- **Fix**: We aim to release a patch within 7 days for critical issues.

### Safe Harbor
We support safe harbor for security researchers. We will not take legal action against you for research and reporting of vulnerabilities as long as you follow this policy and avoid:
- Accessing or modifying data that does not belong to you.
- Degrading the performance of our services (DoS).
